Programming Exercise: Address Book
---

## Setup
1. Install NodeJS
2. Install Grunt
3. Install Dependencies
```bash
npm install
bower install
```
4. Build Jade and Stylus files
```bash
grunt build
```

## Run Server
```bash
node server.js
```

## URL Endpoints

**Landing:**
http://localhost:8080

**People Data:**
http://localhost:8080/api/people
